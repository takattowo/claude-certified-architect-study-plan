# Claude Certified Architect — Foundations Study Plan

Open, forkable study plan for Anthropic's **Claude Certified Architect (Foundations)** exam.

**Live site:** https://takattowo.github.io/claude-certified-architect-study-plan/

## What's inside

- **12-module study plan** (paced for ~3.5 hr/wk over ~20 weeks; condense or stretch as needed)
- **18 anti-patterns cheatsheet** with criticality tags
- **6 scenario walkthroughs** with correct + anti-pattern for every architectural decision
- **Quick reference** — "always wrong" rules for last-minute review
- **Hands-on tasks** per module so concepts stick

## Use it

**Online:** open the live site link above. Searchable, dark mode, mobile-friendly.

**Fork it:**

```bash
git clone https://github.com/takattowo/claude-certified-architect-study-plan.git
cd claude-certified-architect-study-plan
pip install -r requirements.txt
mkdocs serve
```

Then edit `docs/*.md` to track your own progress. Every checkbox is yours to tick.
