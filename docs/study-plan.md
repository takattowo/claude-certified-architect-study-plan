## Pacing options

<div class="grid cards" markdown>

-   :material-rocket-launch: **Sprint**

    12+ hr/week · ~4 weeks · Heavy lift, fastest result.

-   :material-walk: **Standard**

    7 hr/week · 12 weeks · Matches the source plan's calendar.

-   :material-leaf: **Relaxed (default)**

    3.5 hr/week · ~20 weeks · Sustainable alongside a full-time job.

-   :material-coffee: **Light**

    < 3 hr/week · 24+ weeks · Minimum viable consistency.

</div>

Pick a pace. Each module is self-contained — work them in order, but take as long as you need per module.

## How each module is structured

<ol class="ccaf-steps" markdown>
  <li markdown>**Read & note** — concepts to absorb from the domain pages.</li>
  <li markdown>**Hands-on** — code something to anchor the concept in muscle memory.</li>
  <li markdown>**Anti-patterns to memorize** — wrong-answer patterns from the cheatsheet.</li>
  <li markdown>**Verify** — answer the relevant slice of the [source practice question bank](https://claudecertifications.com/claude-certified-architect/practice-questions) (25 total, filterable by domain).</li>
</ol>

You're done with a module when every checkbox is ticked **and** you can explain the anti-patterns in your own words.

!!! info "About the practice questions"
    The source site has **one** practice question bank: 25 questions total, filterable by domain (D1: 6, D2: 4, D3: 5, D4: 5, D5: 5). It's not split into "Practice Test 1, 2, 3…". Each module's verify step points you at the relevant domain filter.
