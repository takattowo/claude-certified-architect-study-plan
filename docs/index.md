## How to use this site

1. **Read the [Exam Overview](exam-overview.md)** — know what you're aiming at (passing score, format, weights).
2. **Work through the [Study Plan](study-plan.md)** — one module at a time. Tick checkboxes as you go.
3. **Drill the [Anti-Patterns](anti-patterns.md)** — these are the highest-leverage memorization. Most wrong answers map to one of these 18.
4. **Study the [6 Scenarios](scenarios.md)** — the exam pulls 4 of 6 at random. Master all of them.
5. **Use the [Quick Reference](quick-reference.md)** — for last-minute review the day before.

## Customize for yourself

Fork the repo. Every checkbox lives in plain markdown. Edit, commit, push — your own personalized tracker on your own GitHub Pages site.

```bash
git clone https://github.com/takattowo/claude-certified-architect-study-plan.git
cd claude-certified-architect-study-plan
pip install -r requirements.txt
mkdocs serve
# edit docs/*.md, commit, push — auto-deploys via GitHub Actions
```
