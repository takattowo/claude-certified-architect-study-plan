## How to use this site

<ol class="ccaf-steps" markdown>
  <li markdown>**Read the [Exam Overview](exam-overview.md).** Know what you're aiming at — passing score, format, weights.</li>
  <li markdown>**Work through the [Study Plan](study-plan.md)** one module at a time. Tick checkboxes as you go.</li>
  <li markdown>**Drill the [Anti-Patterns](anti-patterns.md)** — highest-leverage memorization. Most wrong answers map to one of these 18.</li>
  <li markdown>**Study the [6 Scenarios](scenarios.md).** The exam pulls 4 of 6 at random — master all of them.</li>
  <li markdown>**Use the [Quick Reference](quick-reference.md)** for last-minute review the day before.</li>
</ol>

## Make it yours

Fork the repo. Every checkbox lives in plain markdown. Edit, commit, push — your own personalized tracker on your own GitHub Pages site.

```bash
git clone https://github.com/takattowo/claude-certified-architect-study-plan.git
cd claude-certified-architect-study-plan
pip install -r requirements.txt
mkdocs serve
# edit docs/*.md, commit, push — auto-deploys via GitHub Actions
```
