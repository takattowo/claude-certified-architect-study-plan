# Phase 2 — Applied Knowledge (Modules 5–8)

Focus: Claude Code configuration, plan mode, CI/CD integration, prompt engineering, validation strategies.

By the end of Phase 2 you should be able to:

- Lay out a CLAUDE.md hierarchy for a real team and explain the precedence rules
- Decide command vs skill instantly for any task
- Run Claude Code in CI with structured output and separate-session review
- Build a `tool_use`-based extractor with validation-retry on real data

    ---

    Layered validation (schema/format/semantic/cross-field/external). End-to-end Batches workflow. Multi-pass code review (per-file + cross-file). The cognitive mechanism behind same-session review failure.
